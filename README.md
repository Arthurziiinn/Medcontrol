# 💊 Organizador de Medicamentos

## 📌 Problema
Idosos frequentemente esquecem de tomar medicamentos nos horários corretos.

## 💡 Solução
Aplicação CLI simples para gerenciar medicamentos e horários.

## ▶️ Como executar
python src/app.py

## 🧪 Testes
