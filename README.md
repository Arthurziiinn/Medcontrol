# 💊 Organizador de Medicamentos

## 📌 Problema
Idosos frequentemente esquecem de tomar medicamentos nos horários corretos.

## 💡 Solução
Aplicação CLI simples para gerenciar medicamentos e horários.

## 🌐 Integração com API
A aplicação utiliza a API pública World Time API para mostrar o horário atual em tempo real.

API:
https://worldtimeapi.org/

## ⚙️ Funcionalidades
- Adicionar medicamento
- Listar medicamentos
- Remover medicamento
- Mostrar horário atual

## ▶️ Como executar
```bash
python app.py
```

## 🧪 Como executar os testes
```bash
python -m unittest test_app.py
```

## 📦 Dependências
```bash
pip install -r requirements.txt
```
